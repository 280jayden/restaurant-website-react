# Feane - Restaurant Website (React)

Homework 2 - rebuild of my Homework 1 restaurant site in React, styled with Tailwind CSS.

It's a single-page fast food restaurant site with a hero slider, a filterable menu,
a working shopping cart, an image gallery, an about section and a contact form. Everything
that was plain HTML/CSS/JS in HW1 is now React components with state.

## Run it locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build

```bash
npm run build
```

## Stack

- React (Vite)
- Tailwind CSS
- Font Awesome icons
